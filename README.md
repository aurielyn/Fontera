# Fontera

a very simple font rendering library inspired by how [devonian](https://github.com/Synnerz/devonian/) handles it's font rendering.

# Credits

- [Devonian](https://github.com/Synnerz/devonian/tree/1.21.10)'s font rendering - Basically the inspiration for the whole thing, contributors: DocilElm, PerseusPotter;

# Why create Fontera?

The process to render custom fonts with actual good quality should NOT be as hard as it is.
